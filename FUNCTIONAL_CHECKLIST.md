# 🚀 统一云盘自动转存系统功能清单 (Functional Checklist)

> 基于 139 (移动云盘) 与 Quark (夸克网盘) 的整合重构

## 1. 账号管理模块 (Account Management)
- [x] **多平台支持**：支持移动云盘 (139)、夸克网盘 (Quark)。
- [x] **多账号登录**：支持通过 Cookie、Authorization (139) 进行登录。
- [x] **自动续期**：账号 Token/Cookie 的自动检查逻辑。
- [x] **空间监控**：展示各账号的总空间、已用空间及会员等级。
- [x] **错误提示人性化**：将晦涩的 API 错误码（如 01000004）映射为直观的中文描述。
- [ ] **每日签到**：集成夸克网盘的自动签到功能。

## 2. 任务管理模块 (Task Management)
- [x] **链接识别**：支持直接识别 139 和夸克的分享链接。
- [ ] **子目录支持**：支持选择分享链接中的特定子文件夹进行转存。
- [x] **目标路径选择**：支持通过 WebUI 设置保存位置。
- [ ] **常用目录**：支持收藏常用的保存路径。
- [x] **增量更新策略**：仅转存分享链接中新增的文件。
- [x] **任务生命周期**：支持任务的创建、编辑、删除及手动执行。
- [x] **一键全部运行**：支持过滤掉致命错误任务，批量提交至 Worker 池。
- [x] **错误分类与视觉警告**：识别 [Fatal] 致命错误并自动阻断、UI 变红。

## 3. 自动化与调度引擎 (Automation & Scheduler)
- [x] **Go 高并发调度器**：基于 Goroutine 实现的任务队列。
- [x] **Cron 定时检查**：支持“全局默认+局部重写”的双层 Cron 调度机制，带后端格式校验。
- [ ] **智能重试机制**：针对网络波动实现指数级退避重试。
- [ ] **垃圾清理**：支持定时自动清空移动云盘 (139) 的回收站。
- [ ] **AI Cron 助手**：支持自然语言描述（如“每天下午3点”）生成标准 Cron 语法。

## 4. 整理引擎 (Renaming & Processing Engine)
- [x] **正则重命名整合**：继承夸克项目的正则处理逻辑与魔法变量。
- [x] **动态模板**：利用 Go `text/template` 实现批量重命名模板功能。
- [ ] **AI 智能重命名**：接入大模型，通过自然语言指令完成复杂文件名整理。
- [ ] **规则过滤**：支持通过正则黑白名单过滤特定文件。
- [ ] **层级整理**：支持将多级目录的文件提取并平铺。

## 5. 媒体库与生态集成 (Media & Integrations)
- [ ] **STRM 生成**：自动生成 `.strm` 文件。
- [ ] **Emby/Plex 联动**：转存完成后触发媒体库刷新。
- [ ] **资源搜索集成**：接入 CloudSaver 接口。

## 6. 通知与交互模块 (Communication)
- [ ] **Telegram 机器人**：远程任务控制与搜索。
- [ ] **多渠道消息推送**：集成 Bark、企业微信等。

## 7. Web 管理界面 (User Interface)
- [x] **现代化 Dashboard**：展示已规划任务、容量汇总、近期动态（支持 15 条历史）。
- [x] **原生暗黑模式**：支持深色/浅色主题切换。
- [x] **实时日志流**：网页端通过 SSE (Server-Sent Events) 实时显示系统日志。
- [x] **实时执行状态**：高度对齐的等宽字体监控面板，支持百分比进度与手动忽略。
- [x] **极致同步**：全站状态变更即时推送，清空日志支持全组件联动。

## 8. 系统与安全 (System & Security)
- [x] **单体部署**：后端 Go + 前端 Vue 3 基础结构搭建完成。
- [x] **状态自愈**：服务器启动时自动清理并重置异常中断的 Running 任务。
- [x] **分级结构化日志**：引入 log/slog，支持 LOG_LEVEL 环境变量及双路输出。
- [x] **质量管控与测试**：核心模块（Renamer, Worker, Utils）已实现高覆盖单元测试。
- [ ] **多用户/权限管理**：WebUI 登录鉴权。
- [ ] **数据兼容迁移**：旧版数据迁移工具。
